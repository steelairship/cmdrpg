Command RPG

2020

created by Shyheim Williams

to run:
	run cmdrpg executable


basic commands:
	continue - move to next level
	back - return to the town
	attack - attack the enemy
	heal - restore your health at the cost of mana
	run - try to escape from battle and return to town
	info - show your stats (health, mana, gold, etc)
	help - shows all available commands
	quit - exit the game
	
tips:
	don't be afraid to run away if the enemy is too powerful
	
	sleeping at the inn can get expensive. Remember to heal yourself so your health doesn't deplete as quickly
	
	the temple can grant valuable boons to your health and mana
	
	purchase a better weapon if your attacks are ineffective against enemies
